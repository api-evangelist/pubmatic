# PubMatic GraphQL API

PubMatic is a sell-side platform for digital advertising. Their API covers inventory management, deals and private marketplaces, audience segments, yield analytics, and programmatic direct workflows.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from PubMatic REST API
**Documentation:** https://help.pubmatic.com
